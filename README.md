# Projeto Calculadora de IMC

Este repositório contém um programa Java que calcula o Índice de Massa Corporal (IMC) do usuário, solicitando informações como nome, altura, peso e idade.

## Descrição

O projeto solicita ao usuário que insira seus dados pessoais (nome, altura, peso e idade) e, em seguida, calcula o IMC e exibe o resultado juntamente com as informações fornecidas.

## Estrutura do Projeto

O projeto é composto pelo seguinte arquivo:

* `CalculadoraIMC.java`: Implementa a classe principal que realiza os cálculos do IMC.

## Como Executar

1.  Clone o repositório:

    ```bash
    git clone [https://github.com/mateuscalmeidac/calculadora-imc.git](https://github.com/mateuscalmeidac/calculadora-imc.git)
    ```

2.  Compile o arquivo Java:

    ```bash
    javac CalculadoraIMC.java
    ```

3.  Execute a classe `CalculadoraIMC`:

    ```bash
    java CalculadoraIMC
    ```
